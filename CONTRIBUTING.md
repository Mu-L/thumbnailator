# Contributing to Thumbnailator

Thank you for your interest in contributing to Thumbnailator!

## Pull Requests

Please note that pull requests are not being accepted.

## Bug Reports and Feature Requests

Contributions in the form of bugs reports or feature enhancements are welcome!

But first, please read the [Frequently Asked Questions (FAQ)](https://github.com/coobird/thumbnailator/wiki/FAQ).

If you think you've found a bug or have an idea for a new feature or improvement, please search through [issues](https://github.com/coobird/thumbnailator/issues) first.
Only if it hasn't been reported yet, please file a new issue.
